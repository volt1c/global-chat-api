# Global Chat Api

## Description

Application with public chats without authorization and logs.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Config

Config rooms in [`config/default.json`](https://github.com/volt1c/global-chat-api/blob/main/config/default.json).
